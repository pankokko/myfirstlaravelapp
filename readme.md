<h1>Usage</h1>

<p>新規登録画面で新規登録いただくかログインページ内にある[Login with Git hub」からGithubのアカウントと連携しログインする事が可能です。</p>

<h2>画像投稿</h2>
<p>トップページの左上にある投稿するボタンを押すと画像投稿ページへジャンプします。</p>
<p>最大18Mの容量の画像をアップロードする事が可能です。</p>
<p>投稿した画像はトップページとトップページにある画像を選択した先の詳細ページ<br>
    そして右上のユーザー名をクリックした先のマイページで一覧表示されます。</p>

<img width="1386" alt="スクリーンショット 2020-01-14 16 06 22" src="https://user-images.githubusercontent.com/56859984/72321690-e4ab5380-36e7-11ea-800b-a8e6cc46670c.png">

<p>18Mを超える場合はエラーが表示される</p>

<img width="1339" alt="スクリーンショット 2020-01-10 16 21 10" src="https://user-images.githubusercontent.com/56859984/72133664-5c663f00-33c5-11ea-8119-070f207ebe82.png">


<h2>詳細ページ</h2>
<p>ログインしている場合ページ内にあるコメント欄からコメントを投稿する事ができ、いいねボタンを押す事が出来ます。</p>

<img width="1442" alt="スクリーンショット 2020-01-10 16 04 23" src="https://user-images.githubusercontent.com/56859984/72132819-40619e00-33c3-11ea-9456-74996a41b695.png">

<img width="1429" alt="スクリーンショット 2020-01-14 15 49 55" src="https://user-images.githubusercontent.com/56859984/72320781-b75da600-36e5-11ea-80c2-20cd46ee802e.png">

<p>コメントを投稿したユーザーだけがコメントを削除出来るようになっています。</p>

<h2>アルバム機能</h2>
<p>ヘッダーにあるユーザー名のドロップダウンメニューの「マイページ」を選択すると表示される<br>
「アルバム一覧」と「アルバムを作る」の項目から自身のアルバムを作成と閲覧可能です。</p>


<img width="1449" alt="スクリーンショット 2020-01-10 15 40 44" src="https://user-images.githubusercontent.com/56859984/72131930-0b544c00-33c1-11ea-9b6c-b9c33239c0c2.png">

<p>例：アルバム名「紅葉」の中身を開くと「アルバムに写真を追加」というボタンがあります。</p>
<p>ここからアルバム内に写真を追加する事が出来ます。</p>

<img width="1425" alt="スクリーンショット 2020-01-10 15 43 13" src="https://user-images.githubusercontent.com/56859984/72132345-0b088080-33c2-11ea-84f3-de153d559508.png">

<h2>カテゴリー一覧</h2>
<p>ヘッダー左上の写真を見るを選択するとカテゴリーにマッチした写真を一覧表示するページへとジャンプします。</p>
